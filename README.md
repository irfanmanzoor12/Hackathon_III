# Hackathon III – Reusable Intelligence Platform

This repository contains a complete set of **CAPS playbooks** and documentation for deploying a cloud-native system using Kubernetes and AI-native workflows. It includes:

- FastAPI + Dapr agent
- Next.js frontend
- PostgreSQL state store
- MCP Python server
- Code Runner service
- Docusaurus documentation site

## Getting Started

Follow the `RUN_DEMO.md` guide to deploy and validate the system in under 10 minutes.

## Key Artifacts

- `playbooks/` — Executable deployment playbooks
- `RUN_DEMO.md` — Live validation guide
- `FINAL_SUBMISSION.md` — Judge-ready summary
- `PROJECT_STATUS.md` — Current project status
